# 문서 뷰어 상호작용 디자인

## 1. 문서 열람 상호작용

### 1.1 문서 열기
- 파일 탐색기에서 더블클릭
- 드래그 앤 드롭
- 키보드 단축키: Cmd/Ctrl + O

### 1.2 문서 탐색
- 스크롤을 통한 페이지 이동
- 페이지 썸네일을 통한 빠른 이동
- 검색 기능을 통한 내용 검색

### 1.3 문서 보기 설정
- 확대/축소 (Zoom In/Out)
- 페이지 맞춤 (Fit to Page/Width)
- 회전 기능

## 2. 문서 종류별 특화 기능

### 2.1 증거자료
- 증거자료 메타데이터 표시
- 증거자료 출처 및 입수 경위 표시
- 증거능력/증명력 관련 메모 기능

### 2.2 참고자료
- 참고자료 분류 태그 지정
- 중요도 표시
- 관련 문서 링크

### 2.3 프로젝트최종결정문
- 주문/이유 구분 보기
- 판결요지 하이라이트
- 관련 법령/판례 링크

## 3. 공통 기능

### 3.1 주석 및 메모
- 텍스트 하이라이트
- 스티커 메모
- 그리기 도구

### 3.2 공유 및 협업
- 주석 공유
- 문서 링크 생성
- 변경 이력 추적

### 3.3 문서 관리
- 북마크 기능
- 최근 열람 문서 기록
- 문서 메타데이터 편집
